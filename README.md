# JokerDemolisher - Balatro Joker Ban Tool

## Introduction

JokerDemolisher is a simple tool for the game Balatro that allows users to permanently ban jokers from appearing in any run. Simply click on a joker in the collection to toggle its banned status.

I'm open to suggestions and contributions! Please feel free to create a PR or message me on Discord.

## Requirements

- Balatro 1.0.1o (or newer)
- Steamodded v1.0.0 (or newer)

## How to Use

- Load up modded Balatro (see [Steamodded](https://github.com/Steamopollys/Steamodded))
- Go to **Collection** â†’ **Jokers**
- Click on any joker to ban/unban it
- Banned jokers will flip face-down and won't appear in future runs
- Click again to unban

## Features

- **Click to Ban**: Simply click on any joker in the collection to toggle its banned status
- **Visual Feedback**: Banned jokers appear flipped (face-down) in the collection
- **Persistent Bans**: Your banned jokers are saved and persist across game sessions
- **Works with Modded Jokers**: Compatible with jokers added by other Steamodded mods
